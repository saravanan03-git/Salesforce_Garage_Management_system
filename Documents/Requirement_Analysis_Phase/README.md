# Requirement Analysis Phase - Garage Management System (GMS)

This folder contains the requirement specification and analysis documents for the Garage Management System Salesforce project.

## Files
- **Solution_Requirements.pdf**: Functional and non-functional requirements.
- **Data_Flow_Diagrams_and_User_Flow.pdf**: DFD (Level 0–2), User Flow, and Use Case Diagrams.
- **Technology_Stack_Template.pdf**: Lists Salesforce tools, frameworks, and technologies.
- **README.md**: Summary of requirements and system scope.

### Summary
The Requirement Analysis Phase defines the project’s technical and functional foundations.
It provides detailed insights into user workflows, data exchange, and the Salesforce-based technology stack that supports the GMS solution.
